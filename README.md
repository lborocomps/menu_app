# Recipe App

This is a simple SwiftUI recipe application for iPhone with a TV-style layout.

## Features

- **Daily Recommendation**: The app shows a daily recommended recipe every time you open it.
- **Category Filter**: Filter recipes by `Seafood`, `Vegan`, or `Light` options.
- **Search by Ingredient**: Quickly find dishes that use a specific ingredient (e.g. "fish sauce").
- **Recipe Detail**: Each recipe displays an image, ingredient list, and step-by-step instructions.

## Building

Open the `MenuApp` folder in Xcode and build the project for iOS.

Since the environment here does not include the iOS SDK, compilation is not possible within this container.
